# Bootstrap Dashboard

Dashboard di boolean creata utilizzando il framework Bootstrap 5.

## Contenuti

- Header con barra di ricerca e menu icone con dropdown
- Navbar laterale
- Sezione per aggiungere nuovi studenti con offcanvas 
- Sezione centrale formata da:
1. Tabella prossime classi
2. Sezione F.A.Q. con accordion
3. Sezione Todo con checkbox
4. Sezione utenti attivi con grafico e dati ultimi mesi
